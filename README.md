# Employee Management System

📌 Project Description
This is a full stack web application developed using:
- React.js (Frontend)
- Spring Boot (Backend)
The application performs CRUD operations on employee data.

 🚀 Features
- Add Employee
- Update Employee
- Delete Employee
- View Employee List

🛠️ Technologies Used
- React.js
- Spring Boot
- Java
- MySQL 

📂 Project Structure
- ems-frontend → React Application
- ems-backend → Spring Boot Application

 💡 Functionality
This project allows users to manage employee details efficiently through a user-friendly interface.
